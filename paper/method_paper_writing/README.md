# Method Paper Writing Pack

이 폴더는 `method paper`를 실제로 쓰기 위해 필요한 핵심 정리만 모아둔 곳이다.

원칙:

- 여기에는 `쓰기용 판단`만 둔다.
- 실험 원본 CSV는 복사하지 않고 링크만 둔다.
- 브레인스토밍, 아이디어 확장, 장기 로드맵은 최대한 배제한다.
- 각 문장은 가능한 한 현재 저장소의 증거와 직접 연결된다.
