# Chapter 07: 채널 분해 -- 신호는 어디에 있는가

## 상호작용 효과의 발견과 한계

---

앞 장에서 SNR이 인터페이스 선택을 결정한다는 것을 보았다. 하지만 그것만으로는 **왜** concat이 FiLM을 이기는지, 메커니즘 수준에서 설명되지 않는다. "잡음이 많으면 concat이 유리하다"는 현상 기술이지, 원인 추적이 아니다.

이 장에서는 다른 각도로 접근한다. 예측 신호가 컨디셔닝 채널의 **어디에** 있는지를 분해한다. 채널을 하나씩 제거하고, 짝을 지어보고, F-검정으로 상호작용 항을 식별한다.

결론을 먼저 말하면: 핵심 신호는 개별 채널이 아니라 채널 **간**에 있다. 그리고 이 발견에는 불편한 한계가 따라온다.

---

## 1. 채널 제거 실험

우리의 컨디셔닝 채널은 세 개다:

- **intensity**: 시장 활동 강도 (변동성 기반)
- **position**: 시계열 내 상대적 위치
- **indexret**: 앵커 지수(S&P 500)의 수익률

concat_a 인터페이스에서 채널을 하나씩 제거하며 IC를 측정했다.

### 표 1: concat_a 채널 분해 (GSPC, 2022-2024, 3시드 평균)

| 채널 조합 | IC |
|----------|-----|
| intensity + position + indexret (전체) | 0.057 |
| intensity + indexret | 0.059 |
| intensity + position | 0.057 |
| intensity only | 0.007 |
| position only | 0.019 |
| indexret only | 0.021 |
| 선형 앙상블 (개별 모델 합) | ~0.019 |

이 표에서 세 가지가 보인다.

### 관찰 1: 개별 채널은 약하다

intensity만으로는 IC = 0.007. 거의 static 모델(컨디셔닝 없음) 수준이다. position만으로는 0.019, indexret만으로는 0.021. 개별 채널의 합은 약 0.028이다.

### 관찰 2: 결합하면 개별 합의 2배 이상이다

전체 채널을 결합하면 IC = 0.057. 개별 합(0.028)의 **2배 이상**이다. 선형 앙상블(각 채널로 따로 학습한 모델의 예측을 합산)의 IC가 ~0.019인 것과 비교하면, 결합의 이점이 더 극적으로 드러난다.

이것은 **상호작용 효과(interaction effect)**다. 개별 채널은 약하지만, 특정 채널의 조합이 비선형적으로 강한 신호를 만든다.

### 관찰 3: position은 거의 기여하지 않는다

intensity + indexret (0.059) ≈ 전체 (0.057). position을 제거해도 IC가 떨어지지 않는다. 오히려 미세하게 올라간다. 핵심 상호작용은 intensity와 indexret 사이에 있다.

---

## 2. 증분적 식별: F-검정

채널 분해가 "시너지가 있다"를 보여준다면, 증분적 식별은 "**어떤** 시너지인가"를 식별한다.

Fama-French WML 5일 예측 데이터에서 중첩 회귀 모형의 F-검정을 수행했다.

### 표 2: 중첩 회귀 모형의 F-검정

| 모형 | 사양 | IS R-sq | F vs 이전 | p |
|------|------|---------|----------|---|
| M0 | 상수 | baseline | -- | -- |
| M1 | bear + intensity | +0.001 | -- | -- |
| M2 | M1 + bear x intensity + bear x vix | +0.002 | -- | -- |
| M3 | position + intensity + position x intensity | +0.003 | -- | -- |
| **M4** | **M3 + indexret + intensity x indexret** | **+0.008** | **14.335** | **6.09e-07** |

M3에서 M4로 갈 때, intensity x indexret 상호작용 항을 추가하면 F = 14.335, p = 6.09e-07. 99.99% 신뢰 수준에서 유의하다.

intensity x indexret 상호작용 계수: 0.8884, p = 0.0098.

이것이 말해주는 것: 핵심 예측 신호는 **"고변동성(높은 intensity)이 강한 지수 모멘텀(높은 indexret)과 결합할 때"** 교차 단면 수익률 예측력이 급증하는 패턴이다. 개별적으로는 약하지만 상호작용에서만 강하게 나타나는 시너지적 신호다.

---

## 3. XIP와 concat의 비교

상호작용이 핵심이라면, **명시적으로** 상호작용을 투사하는 인터페이스가 더 좋아야 하지 않은가?

XIP(Explicit Interaction Projection)는 정확히 그렇게 설계했다. 컨디셔닝 채널 간의 상호작용 항을 명시적으로 계산하여 투사한다.

| 인터페이스 | IC |
|-----------|-----|
| XIP | 0.061 |
| concat_a | 0.057 |

XIP(0.061)가 concat(0.057)과 동등하거나 약간 높다. 하지만 **압도적 차이는 없다.**

이것이 시사하는 바: concat은 이미 상호작용을 발견하고 있다. 첫 번째 투사층 `W_proj @ [x; intensity; indexret]`에서 각 뉴런이 intensity와 indexret에 동시에 접근하기 때문이다. 명시적으로 상호작용 항을 만들어 줘도, concat이 자체적으로 학습하는 것과 비슷한 수준이다.

이것은 concat이 이기는 메커니즘에 대한 중요한 단서다. concat이 FiLM을 이기는 이유는 concat이 "더 잘" 학습해서가 아니라, concat이 **조기 융합(early fusion)**을 통해 상호작용에 접근할 수 있기 때문이다.

FiLM에서는 intensity와 indexret가 같은 선형 결합에 나타나지 않는다:

```
FiLM: h = gamma(c) * f(W @ x + b) + beta(c)
```

gamma(c)와 f(W @ x + b)가 곱해지지만, c 내부의 채널 간 상호작용은 gamma 함수가 스스로 학습해야 한다. 상호작용의 발견이 더 깊은 층에서, 더 약한 그래디언트를 통해 일어나야 한다.

## 4. IXIC 재현 시도: 여기서 이야기가 복잡해진다

S&P 500(GSPC)에서 발견한 패턴을 Nasdaq(IXIC)에서 재현하려 했다.

### 표 3: IXIC 채널 분해 결과 (3시드 평균, concat_a)

| 채널 조합 | IXIC IC | GSPC IC (비교) |
|----------|---------|---------------|
| 전체 (모든 채널) | 0.018 | 0.057 |
| intensity + indexret | 0.020 | 0.059 |
| intensity only | 0.005 | 0.007 |
| position only | 0.018 | 0.019 |
| indexret only | 0.023 | 0.021 |
| static (무컨디셔닝) | 0.020 | 0.010 |

GSPC에서 상호작용(intensity + indexret)의 IC는 0.059로, 개별 합(~0.028)의 2배 이상이었다. IXIC에서는? 0.020. 개별 채널과 유사한 수준이다.

**상호작용 효과가 소실되었다.**

### 개별 채널은 유사하다

intensity(0.005 vs 0.007), position(0.018 vs 0.019), indexret(0.023 vs 0.021). 개별 채널의 IC는 두 시장에서 비슷하다. 차이는 **상호작용 패턴에서만** 발생한다.

### static 기준선의 역전

GSPC에서 static IC = 0.010이었으나 IXIC에서는 0.020이다. 컨디셔닝 없이도 IXIC에서 더 높다. IXIC의 교차 단면 구조가 시장 상태와 독립적인 요소를 더 많이 포함한다는 뜻이다.

### 시드 분산

IXIC 결과의 시드 간 IC 범위가 극단적으로 넓다. concat_a 전체: -0.038 ~ +0.061. GSPC(0.023 ~ 0.076)보다 불안정하다. 유효 SNR이 GSPC보다 더 낮을 가능성이 있다.

## 5. 10에폭 안정성: 패턴 자체는 재현된다

IXIC에서 상호작용이 약한 것은 사실이다. 하지만 concat > FiLM이라는 **패턴 자체**는 재현되는가?

IXIC에서 에폭 수를 10으로 확장하고 시드 4개로 검증했다.

결과:
- **IXIC 10에폭:** concat_a IC = 0.073, film_a IC = -0.058
- **t-test p = 0.006**

concat > FiLM 패턴은 IXIC에서도 통계적으로 유의하다(p < 0.01). 높은 시드 분산에도 불구하고, 에폭 수를 늘리면 패턴이 안정적으로 나타난다.

이것이 말해주는 것: **concat의 우위는 시장에 관계없이 재현된다.** 하지만 그 우위의 **메커니즘**(상호작용 효과)은 시장에 따라 다르다. GSPC에서는 intensity x indexret 상호작용이 핵심이지만, IXIC에서는 다른 요인이 작용하고 있을 수 있다.

---

## 6. 왜 시장마다 다른가?

"상호작용이 핵심 신호"라는 발견이 GSPC에 특이적인 이유에 대한 가설:

### 지수 구성의 차이

S&P 500은 대형주 중심이다. 체제 변화(변동성 급등 + 지수 하락)가 교차 단면 수익률에 균일하게 전파된다. 25개 Fama-French 포트폴리오 모두가 비슷하게 반응한다.

Nasdaq은 기술주 집중이다. FAANG/Magnificent 7이 지수 움직임을 지배하지만, 이 움직임이 Size/BM 기반 FF25 포트폴리오의 교차 단면 패턴과 꼭 일치하지 않는다.

### 피측 자산과의 정합

FF25(Size/BM 포트폴리오)는 시가총액과 장부가 대 시가 비율로 구성된다. S&P 500의 대형주 편향과 자연스럽게 정렬되지만, Nasdaq의 기술주 편향과는 불일치한다.

이것은 "intensity x indexret 상호작용"이 **보편적 금융 법칙**이 아니라, **특정 앵커 지수와 피측 자산의 조합에서 나타나는 패턴**이라는 뜻이다.

---

## 7. 정직한 인정

여기서 솔직해야 한다.

Chapter 06에서 "인터페이스 선택은 신호의 속성이 결정한다"고 했다. 이 주장은 유지된다. 하지만 그 다음 단계 -- "핵심 신호는 intensity x indexret 상호작용이다" -- 는 **GSPC에서만 강하고, IXIC에서는 약하다.**

| 주장 | GSPC 증거 | IXIC 증거 | 일반화 가능성 |
|------|----------|----------|-------------|
| concat > FiLM | IC 차이 0.065 | p = 0.006 | **강함** |
| SNR이 인터페이스 선택 결정 | 합성 + 잡음 주입 일치 | (직접 검증 안 함) | 중간 |
| 상호작용이 핵심 신호 | F = 14.335, p = 6.09e-07 | IC 0.020 vs 0.059 | **GSPC 특이적** |
| 조기 융합이 상호작용 접근에 필요 | XIP ≈ concat | (직접 검증 안 함) | 중간 |

논문을 쓸 때 가장 유혹적인 것은 GSPC 결과만 보여주고 IXIC을 생략하는 것이다. "더 강한 스토리"를 위해. 하지만 그러면 다음 사람이 Nasdaq이나 다른 시장에서 재현하려다 실패하고, 우리 논문의 신뢰성이 떨어진다.

그래서 부정적 결과를 포함한다:

> **"상호작용이 핵심 신호"는 S&P 500 앵커에서 확인되었으나 Nasdaq에서 복제되지 않았다. 상호작용 구조는 시장 의존적이다.**

## 8. 상호작용의 의미

GSPC에서 관찰한 상호작용의 경제적 해석:

intensity x indexret가 높다 = **고변동성 + 강한 지수 모멘텀**. 이것은 "시장이 한 방향으로 강하게 움직이면서 동시에 변동성이 높은" 상태다. 공포 국면에서의 급락이나, 유포리아 국면에서의 급등이 이에 해당한다.

이 상태에서 교차 단면 수익률의 예측 가능성이 급증하는 이유: 극단적 체제에서는 팩터 노출이 선명해진다. 평상시에는 잡음에 묻히는 Size/Value 팩터의 교차 단면 패턴이, 시장 스트레스에서 두드러진다.

개별 채널은 왜 약한가:

- **intensity만:** 변동성이 높다는 것만으로는 방향을 모른다. 급등과 급락을 구분 못한다.
- **indexret만:** 시장이 올랐다/떨어졌다는 것만으로는 체제의 강도를 모른다. 조용한 상승과 격렬한 상승을 구분 못한다.
- **둘의 곱:** "시장이 격렬하게 특정 방향으로 움직이고 있다"를 포착한다. 이때 교차 단면 패턴이 선명해진다.

이 해석이 IXIC에서 약한 이유도 설명된다. Nasdaq은 기술주 중심이라 Size/BM 포트폴리오와의 교차 단면 패턴이 S&P 500만큼 깨끗하지 않다.

## 9. 이 장의 발견 요약

| 발견 | 숫자 | 강건성 |
|------|------|--------|
| 개별 채널은 약하다 | IC: 0.007, 0.019, 0.021 | GSPC, IXIC 모두 |
| 결합은 개별 합의 2배 이상 | 0.059 vs 0.028 | GSPC만 |
| intensity x indexret 상호작용이 유의 | F = 14.335, p = 6.09e-07 | WML 데이터 |
| XIP ≈ concat | 0.061 vs 0.057 | GSPC |
| IXIC에서 상호작용 약함 | IC 0.020 vs GSPC 0.059 | 재현 실패 |
| concat > FiLM 패턴은 재현 | IXIC 10에폭 p = 0.006 | 강함 |

핵심 메시지:

1. **예측 신호는 채널 간 상호작용에 있다** -- 적어도 GSPC에서는.
2. **concat이 이기는 이유는 조기 융합으로 상호작용에 접근하기 때문이다.**
3. **이 상호작용 패턴은 시장 의존적이다.** 보편적 법칙이 아니다.
4. **그럼에도 concat > FiLM 패턴 자체는 시장에 관계없이 재현된다.** 상호작용의 구체적 구조가 달라도, concat의 조기 융합 이점은 유지된다.

다음 장에서는 세 번째 발견으로 넘어간다. softmax가 시간 좌표의 신호를 어떻게 파괴하는가.
